# PopcornPicks

PopcornPicks is a small Movie browser using React.

## Getting started

```bash
git clone git@github.com:mejdoubi/popcornpicks.git popcornpicks
cd popcornpicks
npm install
npm run dev
```

## Adding necessary env variable

```bash
echo "VITE_TMDB_API_TOKEN=YOUR_API_TOKEN_HERE" > .env.local
```

## Running tests

```bash
npm run test
```
